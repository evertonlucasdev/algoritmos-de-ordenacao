# Algoritmos de Ordenação em C

Implementações simples de alguns algoritmos de ordenação que estudei durante a disciplina de Projeto e Análise de Algoritmos.

## O que tem aqui

* Quicksort
* Mergesort
* Heapsort

Todos implementados do zero em C, focando em entender como cada um funciona na prática.

## Estrutura

```
algoritmos-de-ordenacao/
 ├── quicksort/
 │    └── quickSort.c
 ├── mergesort/
 │    └── mergeSort.c
 ├── heapsort/
 │    └── heapSort.c
 └── README.md
```

## Como rodar

Usando GCC:

```
gcc quickSort.c -o quickSort
./quickSort
```

(os outros seguem o mesmo padrão)

## Observações

Esse repositório é mais voltado pra estudo mesmo.
A ideia foi praticar:

* recursão
* divisão e conquista
* análise de complexidade
